## My Social Media App

A mobile based social media platform. Share new posts with your followers and public, leave likes and comments on other posts, discover and follow new people.

### Tech Stack

* React Native
* Expo
* MongoDB
* GraphQL
* Apollo Server
* Redis

&nbsp;

- [ ] server: store GraphQL server application
- [ ] app: store React Native Mobile application
